Dialog Registrar Ingreso de Vehículo

DialogRegistrarIngresoVehiculo permite seleccionar un vehículo para registrar su ingreso.

El diálogo presenta los siguientes elementos:

El campo Filtrar permite hacer un filtro de vehículos. Los campos que se tienen en cuenta en la condición del filtro son: el dominio, la marca, el registro interno y el número de chasis.

El ícono PDF para generar un listado impreso con los vehículos.

El listado de vehículos que presenta los vehículos para poder seleccionar.

El botón Registrar Nuevo Ingreso que abrirá una nueva ventana DialogDatosIngresoVehiculo que solicitará los datos necesarios para registrar el ingreso del vehículo.

El botón Hecho cierra la ventana.

Cómo un vehículo?

Para registrar el ingreso de un vehículo, se debe seleccionar un vehículo del listado de vehículos haciendo click sobre alguno de ellos y luego presionar el botón Registrar Nuevo Ingreso. Si se presiona el botón Registrar Nuevo Ingreso sin haber seleccionado un vehículo se abrirá una ventana que le indicará que seleccione un vehículo.
