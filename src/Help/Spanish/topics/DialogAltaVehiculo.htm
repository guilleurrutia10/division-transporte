Dialog Alta Vehículo

Dialog Alta Vehículo permite registrar un nuevo vehículo. Para ello, se debe proporcionar los siguientes datos.:

Dominio
Identificador del vehículo (dominio o patente). Obligatorio.

Marca
Nombre de la marca del vehículo. Obligatorio.

Modelo
Nombre del modelo del vehículo. Obligatorio.

Registro Interno
Identificador interno del vehículo. Obligatorio.

Número de Chasis
Identificador del chasis del vehículo. Obligatorio.

El botón Aceptar para confirmar los datos ingresados y registrar el nuevo vehículo.

El botón Cancelar para cerrar la ventana abortando el registro de un nuevo vehículo.
