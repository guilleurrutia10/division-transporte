Dialog Datos Egreso Vehículo

DialogDatosEgresoVehiculo permite registrar el egreso de un vehículo.

La ventana presenta los siguientes elementos:

Vehículo muestra el dominio del vehículo.

Kilometraje de egreso número que indica la cantidad de kilómetros recorridos en el momento de registrar el egreso.

Nivel de combustible

Fecha de egreso fecha que indica el momento en que se registra el egreso.

El botón Aceptar para registrar los datos ingresados y confirmar el egreso.

El botón Cancelar para cerrar la ventana abortar el registro del egreso.
