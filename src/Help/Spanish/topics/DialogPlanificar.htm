Dialog Planificar

DialogPlanificar permite planificar las reparaciones asignadas de un vehículo.

La ventana presenta los siguientes elementos:

Pestaña Sección

Pestaña Asignar de turnos

Sección muestra el nombre de la sección seleccionada.

Un menú desplegable Fecha para seleccionar el día del turno.

Un menú desplegable Hora para seleccionar la hora del turno.

El listado de reparaciones muestra las reparaciones sin planificar.

El listado de reparaciones asignadas muestra las reparaciones seleccionadas a planificar.

El botón > para agregar reparaciones del listado de reparaciones al listado de reparaciones asignadas.

El botón < para quitar una reparación del listado de reparaciones asignadas.

El botón Crear Turno confirmar el turno. Abre una nueva ventana que informa los datos ingresados acerca del turno registrado.

El botón OK para para registrar las reparaciones planificadas.

El botón Cancel para cerrar la ventana y abortar todas las reparaciones registradas anteriormente.

Cómo planificar reparaciones?

Para registrar la planificación de las reparaciones se debe seleccionar la sección del menú desplegable ... haciendo click sobre alguno de ellos. Tras presionar sobre alguna de las secciones pasaremos a la Pestaña Asignar de turnos que nos muestra día y horario de los turnos no asignados. Aquí se debe seleccionar un día para el turno del menú desplegable Fecha haciendo click sobre alguno de ellos y la hora del menú desplegable Hora haciendo click sobre alguno de ellos. Luego de seleccionar el día y horario, se deben seleccionar las reparaciones* que se van a llevar a cabo en el turno seleccionado (día y hora). Para registrar el turno se debe presionar el botón Crear Turno que abrirá una venatana que informa los datos ingresados acerca del turno registrado. Por útlimo, se debe presionar el botón OK para registrar todas las reparaciones como planificadas, este abrirá una nueva ventana que permitirá seleccionar la ruta y el nombre de la hoja de ruta que genera de los turnos creados.

Nota: Asegurarse de planificar todas las reparaciones antes de presionar el botón Aceptar.

*Las reparaciones se deben cargar una por una. Es decir, se debe seleccionar una reparación para el turno haciendo click sobre alguna de ellas y presionar el botón >. Tras presionar el botón > la reparación seleccionada será visualizada en el listado de reparaciones asignadas que se muestra a la derecha y removida del listado de reparaciones que se muestra a la izquierda. Esto es igual si se quieren seguir agregando reparaciones al turno.
