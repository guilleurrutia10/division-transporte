Tooltip

Tooltip se trata de un texto de ayuda.

Este texto de ayuda se puede ver cuando el usuario pasa el puntero del mouse sobre un elemento de la interfaz gráfica, sin hacer click en él, un texto de ayuda puede aparecer con información acerca del elemento sobre el cual se encuentra posicionado el puntero.
