Login

Login permite autenticación de los usuarios a través del nombre de usuario y la contraseña.

La ventana presenta los siguientes elementos:

Usuario Nombre de usuario.

Contaseña Contraseña de usuario.

El botón Ok para confirmar los datos ingresados. Si los datos ingresados son correctos se habilitan las funcionalidades habilitadas para el usuario.

El botón Cancel para cerrar la ventana.
