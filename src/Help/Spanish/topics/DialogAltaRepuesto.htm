Dilog Alta Repuesto

DialogAltaRepuesto permite registrar un nuevo repuesto, para lo cual, se deben suministrar los siguientes datos obligatorios:

Nombre Repuesto
Se trata de una cadena que permite denominar de alguna manera el repuesto en la aplicación.

Descripción Repuesto
Se trata de una cadena que permite describir el repuesto en la aplicación.

El botón Aceptar para confirmar los datos y registrar un nuevo repuesto.

El botón Cancelar para cerrar la ventana y abortar el registro de un nuevo repuesto.
