Funcionalidades

El objetivo de este capítulo es describir brevemente los temas acerca de las funcionalidades que ofrece la aplicación DivT.

Existen diferentes perfiles en la aplicación. Cada perfil habilita las funcionalidades necesarias que el usuario con dicho perfil puede acceder. Estos son:

Perfiles
Jefe de Divisón
Jefe de Sección
Empleado Administrativo
Inspector

Las funcionalidades se encuentran divididas por diferentes

  • Personal
  • Repuestos
  • Sección
  • Verificación y Reparación
  • Archivo
