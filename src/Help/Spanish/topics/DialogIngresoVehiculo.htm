Dialog Ingreso Vehículo

DialogRegistrarIngresoVehiculo permite a los usuarios registrar el ingreso de un vehículo. Para ello, se debe ingresar los datos que se mencionan a continuación:

Kilometraje actual
Número que indica la cantidad de kilómetros con los cuales se registra el ingreso del vehículo.

Nivel de combustible actual
Se trata de un valor que representa el nivel de combustible en el tanque de nafta con el cual se registra el ingreso del vehículo. Este puede tomar los siguientes valores: LL (lleno), 3/4 (tres cuartas partes), 1/2 (medio), 1/4 (una cuarta parte) y V (vacío).

Comisaría
Comisaría a la cual pertenece el vehículo.

Localidad
Localidad del vehículo.

Chofer asignado
Nombre del chofer a cargo que transporta al vehículo.

Equipamiento
Si el vehículo cuenta con algún equipamiento especial, en este campo se lo debe ingresar. Opcional.

Reparación solicitada
Si se conoce de antemano la/s reparación/es que el vehículo necesita, en este campo se lo debe ingresar. Opcional.
