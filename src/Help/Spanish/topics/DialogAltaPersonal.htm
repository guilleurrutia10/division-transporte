Dialog Alta Personal

DialogAltaPersonal permite registrar un nuevo empleado. Para ello, se debe proporcionar los siguientes datos:

Nombre. Obligatorio.

Apellido. Obligatorio.

Tipo de Documento Valor que puede tomar uno de los siguientes valores: D.N.I (Documento Nacional de Identidad), L.C (Libreta Cívica), L.E (Libreta de Enrolamiento) o C.I.

Número de Documento Identificador numérico del empleado. Obligatorio.

Fecha de Nacimiento Fecha. Obligatorio.

Domicilio Dirección del domicilio. Opcional.

Teléfono Número telefónico. Opcional.

Email Dirección de correo electrónico. Opcional.

El botón Aceptar para confirmar los datos ingresados y registrar el nuevo empleado.

El botón Cancelar para cerrar la ventana y abortar el registro de un nuevo empleado.
