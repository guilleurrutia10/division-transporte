Widget Mostrar Vehículos sin Planificar

WidgetMostrarVehiculosSinPlanificar permite seleccionar un vehículo para planificar sus reparaciones asignadas.

La ventan presenta los siguientes elementos:

El campo Filtrar permite hacer un filtro de vehículos. Los campos que se tienen en cuenta en la condición del filtro son: el dominio, la marca, el registro interno y el número de chasis.

El ícono PDF para generar un listado impreso con los vehículos.

El listado de vehículos que presenta los vehículos para poder seleccionar.

El botón Seleccionar que abrirá una nueva ventana DialogPlanificar que solicitará los datos necesarios para planificar las reparaciones asignadas del vehículo.

Cómo seleccionar el vehículo a planificar?

Para seleccionar el vehículo para planificar sus reparaciones asignadas se debe seleccionar un vehículo del listado de vehículos y luego, presiona el botón Seleccionar. Tras haber presionado se abrirá una nueva ventana DialogPlanificar para planificar las reparaciones.
