Dialog Modificar Vehículo

DialogModificarVehiculo permite registrar la modificación de datos de un vehículo ya registrado.

Los datos que se pueden modificar de un vehículo son los siguientes:

Marca Nombre de la marca del vehículo.

Registro Interno Identificador interno del vehículo.

Número de Chasis Identificador del chasis del vehículo

El botón Aceptar para confirmar los cambios realizados sobre los datos del vehículo.

El botón Cancelar para cerrar la ventana sin realizar ningún cambio sobre los datos del vehículo.

Nota. Si no se realiza ningún cambio sobre los datos y se presiona el botón Aceptar se abrirá una ventana que indicará tal suceso.
