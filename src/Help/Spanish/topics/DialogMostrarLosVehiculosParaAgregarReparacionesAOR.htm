Dialog Mostrar los Vehículos para agregar Reparaciones a la Orden de Reparación

DialogMostrarLosVehiculosParaAgregarReparacionesAOR permite seleccionar un vehículo para registrarle reparaciones.

El diálogo presenta los siguientes elementos:

El campo Filtrar permite hacer un filtro de vehículos. Los campos que se tienen en cuenta en la condición del filtro son: el dominio, la marca, el registro interno y el número de chasis.

El ícono PDF para generar un listado impreso con los vehículos.

El listado de vehículos que presenta los vehículos para poder seleccionar.

El botón Agregar Reparaciones a Orden que abrirá una nueva ventana DialogRegistrarReparaciones .

El botón Hecho cierra la ventana.

Cómo seleccionar un vehículo?

Para registrar las reparaciones de un vehículo, se debe seleccionar un vehículo haciendo click sobre alguno de ellos y presionar el botón Agregar Reparaciones a Orden que abrirá una nueva ventana para registrar las reparaciones del vehículo. Si se presiona sobre el botoón Agregar Reparaciones a Orden y no se selecciona un vehículo se abrirá una ventana indicando el suceso.
