Dialog Agenda de Sección

DialogAgendaDeSeccion permite generar un reporte de la tabla de turnos de una sección para un día o para un rango de días.

La ventana presenta los siguientes elementos:

Día muestra un calendario para seleccionar el día.

Por rango tildar para habilitar los elementos que se presentan a continuación:

Desde día a partir de donde comenzar para generar la tabla de turnos.

Hasta día hasta donde generar la tabla de turnos.

El botón Aceptar para confirmar la generación de reporte/s.

El botón Cancelar para abortar la generación de reporte/s.
