Dialog Registrar Recepción de Pedido de Actuación

DialogRegistrarRecepcionDePedidoDeActuacion permite a los usuarios seleccionar el pedido de actuación para registrar la fecha de recepción.

El campo Filtrar permite realizar un filtro de los pedidos de actuación por el número de pedido.

Filtrado
Permite buscar .....

Listado de pedidos de actuación muestra los pedidos de actuación realizados que no han registrado fecha de recepción.

Botón Registrar Recepción abre una nueva ventana DialogAsignarFechaRecepcionPedidoActuacion para asignar la fecha de recepción.

Botón Hecho para cerrar la ventana.

Cómo ?

Para registrar la recepción del pedido de actuación se debe seleccionar un pedido de actuación, luego, presionar el botón Registrar Recepción. Esto abrirá una nueva ventana para asignar la fecha de recepción.
