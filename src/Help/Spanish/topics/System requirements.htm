Requisitos del sistema

DivT es una aplición distribuida cliente/servidor multiplataforma.

El servidor es el punto de acceso a la Base de Datos, por lo cual su tarea consta de permanecer escuchando y respondiendo las solicitudes de las máquinas Clientes de acceso a la misma.

Los Clientes son ordenadores que corren una aplicativo que les ofrece todas las funcionalidades de la aplicación DivT y que se comunica con el Servidor para solicitar operaciones con la Base de Datos.

Más información acerca de la instalación puede encontrase en Instalación help topic.


Hardware:
Para el cliente:
IBM-compatible PC
256 MB RAM
  "X" MB espacio libre en disco
Para el servidor:
Sistema Operativo:
Microsoft Windows 2000/XP/Vista/7
Linux
Otros Requerimientos:
Conexión de red LAN. Para poder administrar las conexiones entre los Clientes y el Servidor.
