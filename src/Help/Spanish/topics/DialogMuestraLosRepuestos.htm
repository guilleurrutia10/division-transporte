Dialog Muestra los Repuestos

DialogMuestraLosRepuestos permite seleccionar un repuesto para poder modificar sus datos.

La ventana presenta los siguientes elementos:

Buscar Nombre permite hacer un filtro de los repuestos. Los campos del listado que se tienen en cuenta en la condición de filtro son código, nombre ydescripción.

El listado de repuestos muestra todos los repuestos registrados.

El botón Modificar Repuesto abrirá una nueva ventana DialogModicarRepuesto que permitirá modificar los datos de un repuesto.

El botón Cancelar para cerrar la ventana y abortar la selección de un repuesto.

Cómo seleccionar un repuesto?

Para seleccionar un repuesto para poder modificar sus datos se debe seleccionar un repuesto del listado de repuestos haciendo click sobre alguno de ellos y presionar el botón Modificar Repuesto. Tras presionar el botón Modificar Repuesto se abrirá una nueva ventana DialogModicarRepuesto que permitirá modificar los datos de un repuesto.
