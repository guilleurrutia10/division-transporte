Dialog

Un Diálogo (Dialog) es una ventana de nivel superior utilizada principalmente para tareas y comunicaciones con el usuario. El diálogo bloquea la entrada a otras ventanas visibles en la aplicación.

Todas las ventana de diálogo cuentan con un botón X en la esquina superior derecha que permite cerrar la misma.
