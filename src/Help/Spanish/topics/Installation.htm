Instalación

DivT es una aplición distribuida cliente/servidor.

DivT se distribuye a través de... Para tener información acerca de los requerimientos de hardware (ver Requisitos del sistema).

Cómo instalar?

- Cliente

Para correr la aplicación se necesita contar con lo siguiente:

- Python 2.7: Interprete de código python. Disponible en Python

- Paquetes de python:

- PyQt4

- matplotlib

- ZODB

- ZEO

Estos paquetes pueden ser instalados utilizando la herramienta pip, gestor de paquetes de python, easy_install en Windows, o descargando directamente los fuentes de cada paquete las paqinas web oficiales correspondientes.

- En Windows:

- Código fuente de la aplicación: Copiar el código fuente del proyecto en la carpeta C:\ProgramFiles\DivisionTransporte. Mover el acceso directo a la aplicación “División Transporte” al escritorio.

- En Linux:

- Código fuente de la aplicación: Copiar el código fuente del proyecto en la carpeta ~/DivisionTransporte (~, carpeta de usuario. Ejemplo /home/usr/). Ejecutar el archivo main.py de la siguiente manera python main.py.

- Servidor

Para correr la aplicación se necesita contar con lo siguiente:

- Correr el script runzeo que provee el paquete ZEO.

En la carpeta raíz del proyecto se provee un ejecutable que corre el srcipt. Uno para cada plataforma.

Para más información sobre como ejecutar el script runzeo en su sistema puede revisar el código runzeo.py.
